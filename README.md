# ChristianTranslator777

Этот проект представляет собой Telegram-бота-переводчика, использующего API DeepL. Бот предназначен для приватного использования и принимает сообщения только от пользователей, указанных в переменной `ALLOWED_USERS`. После запуска бот предлагает выбрать направление перевода (EN ↔ RU или EN ↔ RO) и переводит присланный текст.

## Переменные окружения

- `BOT_TOKEN` — токен вашего Telegram-бота.
- `DEEPL_API_KEY` — ключ доступа к DeepL API.
- `DEEPL_API_URL` — URL эндпойнта DeepL.
- `ALLOWED_USERS` — список разрешённых ID пользователей Telegram, разделённых запятыми.
- `WEBHOOK_URL` — полный URL для вебхука бота.
- `PORT` — порт, на котором запустится приложение.

## Запуск

```bash
python translate_bot.py
```

Перед запуском убедитесь, что заданы переменные окружения и указан публичный `WEBHOOK_URL`. При необходимости измените `PORT`, на котором будет слушать приложение.

### Рекомендации по настройке сервиса на Render

1. Создайте новый Web Service и загрузите репозиторий.
2. Укажите в настройках окружения значения `BOT_TOKEN`, `DEEPL_API_KEY`, `DEEPL_API_URL`, `ALLOWED_USERS`, `WEBHOOK_URL` и `PORT`.
3. В качестве команды запуска укажите `python translate_bot.py`.
4. Выберите подходящий тип инстанса и план оплаты/бесплатный вариант.

## Ограничения использования

Бот ведёт подсчёт переведённых за месяц символов и сохраняет их в файле `usage.json`.
При достижении лимита в 500 000 символов переводы блокируются до начала
следующего месяца.
