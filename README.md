# Christian Translator Bot

Этот проект представляет собой телеграм-бота для перевода текста с использованием DeepL.

## Быстрый старт

1. Скопируйте файл `.env.example` в `.env`:
   
   ```bash
   cp .env.example .env
   ```
2. Откройте `.env` и заполните значения переменных.

После этого вы можете запускать бота, как указано в коде.
